
# 🍴 Restaurant Management System

A full-stack CRUD web application built with **Node.js, Express, MySQL**, and a styled **HTML/CSS frontend**.  
This project allows restaurant owners to manage menu items, customers, and orders with a sleek dark-themed interface.

---

## ✨ Features
- **Menu Management**: Add, update, delete, and view menu items.
- **Customer Management**: Add, update, delete, and view customers.
- **Order Management**: Place new orders, update quantities, delete orders.
- **Order Details**: View joined data showing which customer ordered which item, quantity, and date.
- **Dark Theme UI**: Stylish design with restaurant background image.

---

## 🛠 Tech Stack
- **Backend**: Node.js, Express
- **Database**: MySQL
- **Frontend**: HTML, CSS, JavaScript
- **Styling**: Custom dark theme with background image

---

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- MySQL installed and running

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/janvikhaire/restaurant-management-system.git
   ```
2. Navigate into the project folder:
   ```bash
   cd restaurant-management-system
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Configure your database connection in `db.js`.

### Run the Project
```bash
node index.js
```
Open your browser at:
```
http://localhost:3000
```
